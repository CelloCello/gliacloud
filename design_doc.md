# 咖啡廳分享/評分平台


## back-end

### 內容
- 登入系統 (token)
- 咖啡廳資訊發布頁面
	* 咖啡廳資訊
- 首頁
	* 搜尋?
	* 最新分享?
- 咖啡廳資訊瀏覽頁面
	* 資訊內容
	* 評分系統
	* google map 串接以顯示位置
- 通知
	* 通知 bar, 類似 facebook 右上角那個
	* 信件


### 會有

- restful api
	* json
	* web (SPA), app
- database
	* mongoDB
- auth
	* token
	* facebook login


### 架構

- api controller
	* 實做 api 與 model 溝通
	* facebook 等等的第三方平台溝通
- model
	* django ORM
